# PROJETO DE TCC
## Criação de uma rede neural de 3 camadas para a identificação e plot dos digitos numericos (0 e 9) escritos a mão.

Esse código foi criado usando o Jupiter notebook para a monografia de fim de curso do aluno Antonio Wagner Rodrigues de Lima para a Universidade federal da Paraíba.
Com o intuito do mesmo aprender um pouco mais sobre as rede neurais artificiais e suas aplicações. Esta aplicação foi retirada do livro de Taric Rashid - Make your own Neural Network.

Logo a RNA criada tem somente uma camada oculta e com performance de 97% de sucesso, utilizando uma coleção de dados com 60 mil arquivos para o treinamento da rede neural, e 10 mil arquivos para teste.
